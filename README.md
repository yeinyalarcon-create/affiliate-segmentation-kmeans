Affiliate Segmentation using K-Means
Business Problem

The objective of this project is to segment affiliates based on behavioral and demographic variables in order to identify homogeneous groups and support strategic decision-making.

Methodology

This project applies K-Means clustering, using the Elbow Method and Silhouette Score to determine the optimal number of clusters.

Tools Used

SQL (data extraction and preparation)

R (data preprocessing and clustering)

K-Means algorithm

Elbow Method

Silhouette Score

Data Disclaimer

The original dataset is confidential. A synthetic dataset with similar structure may be used for demonstration purposes.
